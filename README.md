# NextWatch!! Recommender System
A Content-Based Recommender System for Movies, Anime, and TV Shows using Python, Scikit-Learn, and Streamlit.

## Features
- Filter by Content Type (Movie/Anime/TV)
- Direct TMDB Links for every recommendation
- Data processed from TMDB and Anime datasets

## How to Run
1. Clone the repo
2. Run the Jupyter Notebook `NextWatch!!.ipynb` to generate `similarity.pkl`
3. Run `streamlit run app.py`
